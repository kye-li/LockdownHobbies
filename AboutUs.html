About This Website

This website was created by Chloe Allan and Kye-Li Tan as part of the Code First Girls Intro to Web Development course which took place from October to December 2020.

During this time a second lockdown occured, so it was agreed to create a website which gave advice and tutorials on a variety of hobbies which other people may benefit from during this time.
